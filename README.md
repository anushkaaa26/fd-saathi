# 💰 FD Saathi — Vernacular Fixed Deposit Advisor

FD Saathi is a user-friendly web application designed to help users understand and invest in Fixed Deposits (FDs) in India. It is built with a focus on vernacular users, supporting Hindi, Bhojpuri, and English for better accessibility.

---

## 🌐 Live Demo
👉 https://anushkaaa26.github.io/fd-saathi/

---

## ✨ Features

- 🧮 **FD Calculator**  
  Calculate maturity amount, interest earned, and returns instantly.

- 🏦 **Bank Comparison**  
  Compare FD interest rates across major Indian banks.

- 💬 **Chatbot UI (Demo Mode)**  
  Interactive chatbot interface for financial queries (currently uses simulated responses, ready for AI integration).

- 🌍 **Multi-language Support**  
  Supports Hindi, Bhojpuri, and English for wider reach.

- 🎙️ **Voice Interaction UI**  
  Simulated voice input experience for accessibility.

- 📊 **FD Laddering Visualization**  
  Helps users understand smart FD investment strategies.

- 📑 **Document Checklist**  
  Tracks required documents for FD booking.
